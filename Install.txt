======================================================================
Installation
======================================================================

First, unpack the zip-file and copy or link the TOM8 folder to the spm8/toolbox directory. You can start the toolbox by choosing "TOM8" in the "Toolboxes" selector. Alternatively you can use the "TASKS" button (right in the Graphics window): select TASKS | Tools | Template-O-Matic.
